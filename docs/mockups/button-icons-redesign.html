🎨 按钮图标重设计方案

为文件上传、收起状态面板、回溯三个按钮设计更适配的图标

文件上传按钮 (Add Attachment)
图标方案对比 当前: 回形针
上传 + 加号
清晰表达"添加文件"的含义
现代回形针
经典设计，加圆点装饰
文件 + 加号
明确表示添加文件
图片 + 加号
适合图片上传场景
推荐方案：上传 + 加号
上传箭头配合加号，直观表达"添加附件"的操作意图。比传统回形针更现代，且语义更清晰。
收起状态面板按钮
图标方案对比 当前: 下箭头/图层
面板折叠
底部面板向上折叠
侧边面板
右侧面板向左收起
收起折叠
通用收起图标
最小化
四角收缩效果
推荐方案：面板折叠
矩形面板 + 底部分割线 + 折叠箭头，清晰表达"收起底部面板"的意图。状态面板是从底部展开的，这个图标最符合实际交互。
回溯按钮 (Rewind)
图标方案对比 当前: 撤销箭头
时钟回溯
时钟 + 中心点，表示时间回溯
逆时针旋转
经典的撤销/回退图标
步回退
带竖线的回退，表示回到起点
版本回退
类似 Git revert 的设计
推荐方案：时钟回溯
时钟图标配合中心圆点，直观表达"回到过去某个时间点"的概念。比单纯的撤销箭头更适合"回溯对话历史"的场景。
