方案二 · Bouncy Elastic

活力弹性

所有动效以弹性、活力为核心，使用过冲回弹曲线让 UI 元素"弹"进视野。 每次交互都有明确的物理反馈，适合追求趣味性和操作手感的场景。

🔘 按钮动效

hover 时 scale(1.04) 放大，active 时 scale(0.92) 缩小 — 明显的弹性按压感。支持涟漪水波效果。

设计要点：
• 放大/缩小使用弹性曲线，视觉上像"橡皮"
• 涟漪效果使用真实的圆形 DOM 扩散
• 加载态使用 3 个弹跳圆点，有明确的节奏感
• 按压时过渡缩短到 100ms，释放后恢复

🃏 卡片动效

hover 时整体 scale(1.03) + 上浮 2px，图标旋转 -8° + scale(1.1)，active 时回缩 scale(0.98)。
