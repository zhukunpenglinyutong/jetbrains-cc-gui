无竖线方案的优势

  • 与消息卡片的圆角风格更统一，视觉更柔和
  • 光球脉冲动画比波形更精致，辨识度更高
  • 背景呼吸光晕增加层次感但不抢眼
  • 颜色语义清晰：蓝=生成 / 紫=思考 / 橙=排队
  • 可选波形变体，满足不同审美偏好

设计规范

生成中
  • 蓝色主色调 #4ea1ff
  • 背景 6% 透明度
  • 边框 15% 透明度
  • 光球脉冲 + 扩散光环
  • 8px 圆角
思考中
  • 紫色主色调 #a989ff
  • 与思考区颜色一致
  • 光球脉冲 + 扩散光环
  • 可选波形变体
排队中
  • 橙色主色调 #e7b85a
  • 与警告色系一致
  • 显示预计等待时间
  • 光球脉冲动画
动画参数
  • 光球脉冲: 2s ease-in-out infinite
  • 扩散光环: 2s ease-out infinite
  • 波形动画: 1.2s ease-in-out
  • 呼吸光晕: 3s ease-in-out
  • reduce-motion: 关闭所有动画
