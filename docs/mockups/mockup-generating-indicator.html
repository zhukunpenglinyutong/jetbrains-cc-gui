方案 A：流光胶囊 + 弹跳圆点

Option A

在现有胶囊基础上增加微光流动效果，用三点弹跳替代旋转圆环，更轻盈现代。

正在生成响应 12s

设计要点

  • 胶囊背景使用渐变 + shimmer 流光动画，增加科技感
  • 三点弹跳替代旋转 spinner，视觉更轻巧
  • 毛玻璃 backdrop-filter 增加层次感
  • 保持现有尺寸和布局，改动最小

方案 B：极简内联 + 音频波形

Option B

去掉胶囊边框，用极简的音频波形动画暗示"AI 正在说话"，最不抢眼。

正在生成响应 12s

设计要点

  • 无边框无背景，融入消息流中，最不干扰阅读
  • 四条竖线音频波形动画，暗示"AI 正在输出"
  • 适合消息密集的场景，不会打断视觉流
  • 缺点：视觉存在感较弱，可能被忽略

方案 C：进度环 + 胶囊

Option C

用 SVG 进度环替代旋转圆环，暗示任务进度感（非真实进度），视觉更精致。

正在生成响应 12s

设计要点

  • SVG 环形进度动画，stroke-dashoffset 往返模拟加载
  • 比纯旋转圆环更有"在做某事"的暗示
  • 适合需要强调"任务进行中"的场景
  • 缺点：用户可能误以为是真实进度

方案 D：脉冲光球（推荐）

Recommended

一个会呼吸发光的圆球 + 扩散光环，科技感强且辨识度高，可扩展颜色语义。

设计要点

  • 光球脉冲 + 双层扩散光环，视觉层次丰富
  • 蓝色=生成中 / 紫色=思考中 / 橙色=排队中 / 绿色=完成——颜色语义清晰
  • 动画柔和不刺眼，长时间显示也不会烦躁
  • 与 JetBrains IDE 暗色主题风格契合
  • 可选：光球点击展开显示详细 token 统计

方案 E：打字机效果

Option E

用打字机逐字显示 + 闪烁光标，暗示"文字正在被生成"，概念最直观。

正在生成响应 · 12s
