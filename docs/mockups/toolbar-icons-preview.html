方案C 设计详解

  • 底部左侧工具栏整体容器化：所有功能按钮包裹在一个 background: rgba(0,0,0,0.2); border-radius: 18px 的深色圆角胶囊容器中
  • 容器内去边框 + 透明背景：按钮 hover 时才显示 rgba(255,255,255,0.08) 半透明反馈
  • 细线分隔：Config | Provider 和 Mode | Model 之间用 1px rgba(255,255,255,0.06) 分隔线划分功能区
  • 模式图标组：Chat / Task / Agent / Auto 四个图标平铺在容器内，当前选中的 Agent 以 蓝色半透明背景 高亮
  • 增强按钮：30x30 圆形，橙色主题，hover 时缩放 + 闪烁动画
  • 发送按钮：30x30 圆形渐变 + 内阴影 + 外发光阴影，视觉焦点最突出
  • Tooltip：hover 时显示功能名称，替代文字标签，保持紧凑

响应式策略

  • 宽屏 (>520px)：显示图标 + 文字标签（Sonnet、High），完整功能区
  • 窄屏 (<380px)：自动切换为纯图标模式，文字隐藏，所有按钮退化为 icon-only
  • 容器化优势：左侧整体胶囊容器在缩小时自动压缩内部间距，不会导致布局断裂
  • Mode 组平滑退化：四模式图标从带间距变为紧凑排列，无需额外 media query
设计规范 DESIGN TOKENS

左侧胶囊容器

  • background: rgba(0,0,0,0.2)
  • border-radius: 18px
  • padding: 2px 3px
  • 内按钮 hover: rgba(255,255,255,0.08)
  • 模式高亮: rgba(0,120,212,0.14)

右侧操作按钮

  • 发送按钮: 30x30, 圆形, linear-gradient(#0078d4, #005a9e)
  • 增强按钮: 30x30, 圆形, rgba(245,166,35,0.08)
  • 外发光: box-shadow: 0 2px 8px rgba(0,120,212,0.35)
  • hover 缩放: transform: scale(1.08)
  • 动画曲线: cubic-bezier(0.4, 0, 0.2, 1)
