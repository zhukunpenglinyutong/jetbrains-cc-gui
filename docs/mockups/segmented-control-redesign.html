5. 设计说明

统一设计语言：
• 所有分段控件共享同一套 CSS：.seg-group（容器）+ .seg-btn（按钮）+ .seg-btn.active（选中）
• 图标按钮共享：.icon-btn（通用）+ .provider-btn（紫色变体）
• 圆角统一：分段控件外层 7px，内层 5px
• 间距统一：分段组之间用 3px 圆点分隔符 .sel-sep

响应式策略：
• ≥ 600px：所有文字完整显示
• 400-600px：Mode/Model 隐藏文字，仅显示图标
• ≤ 400px：Model/Reasoning 也折叠为单图标 + ▼ 下拉

CSS 变量：
• 选中背景：rgba(0,120,212,0.15)（暗色）/ rgba(0,120,212,0.08)（亮色）
• 容器边框：rgba(255,255,255,0.08)（暗色）/ rgba(0,0,0,0.08)（亮色）
• 未选中文字：#666 / 选中文字：#e0e0e0
