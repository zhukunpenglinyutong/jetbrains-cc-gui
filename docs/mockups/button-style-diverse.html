🎨 底部按钮个性样式方案

8 种独立按钮风格，替代分段控件，适配插件底部工具栏

样式方案一览（矩阵对比）

A — Glass Chip 玻璃芯片 推荐
半透明背景 + 1px 边框 + backdrop-filter。选中态用 accent 色调。
适合现代玻璃态 UI，与 VS Code 侧边栏风格一致。
B — Neon Glow 霓虹发光 个性
纯文字 + 选中时底部发光线条 + text-shadow。
赛博朋克风，适合深色主题为主的插件。
C — Subtle Lift 微浮起 推荐
hover 时 translateY(-1px) 微浮起 + 选中态加柔和阴影。
物理质感，交互反馈明确，适合工具密集型界面。
D — Tag Label 标签 轻量
未选中虚线边框，选中实线 + accent 背景。
小巧精致，适合空间紧凑的工具栏。
E — Dot Indicator 圆点 个性
选中项左侧加发光圆点，无边框无背景。
Telegram / Slack 风格，极简但辨识度高。
F — Rounded Pill 药丸 个性
border-radius: 13px 全圆角药丸。
Notion / Linear 风格，现代感强，辨识度高。
G — Minimal Text 极简 轻量
零背景零边框，选中仅底部 1.5px accent 线条。
类似浏览器 Tab 风格，最极简。
H — Outlined Badge 描边徽章 推荐
未选中 1px 实线边框，选中 accent 边框 + 微背景。
经典 macOS 风格，清晰专业，改动最小。
