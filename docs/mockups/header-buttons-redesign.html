4. 变更总结

核心原则：所有按钮统一风格（颜色、尺寸、hover 效果完全一致），仅通过图标本身的语义区分功能。

变更项 旧 新
按钮圆角 4px（方正） 6px（微圆，更现代）
stroke-width 1.8（偏细） 2（略粗，小尺寸更清晰）
Hover 效果 微弱背景 背景 + 提亮至 #e8e8e8
Active 反馈 无 scale(0.9) 按压缩小
PlusIcon 裸加号（语义模糊） 文档 + 加号（"新建文件"语义明确）
HistoryIcon 纯时钟（和 ClockIcon 重复） 回旋箭头 + 时钟指针（"回看历史"语义明确）
SettingsIcon 复杂齿轮（30+锚点，18px 下糊成一团） 三滑块（9 条线，18px 下清晰，"调节设置"语义明确）
按钮颜色 #858585 #999（所有按钮完全统一，无特殊色）
