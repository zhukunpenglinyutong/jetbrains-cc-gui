6. 设计说明

改动范围：
- 移除 外层容器 (border + background + "50%" 文字标签)
- 移除 虚线高亮环
- 新增 纯 SVG 圆环，内嵌百分比数字
- 新增 三色分级：绿色 (<50%) → 黄色 (50-80%) → 红色 (>80%)
- 新增 hover 时浅色背景反馈
- 保持 Tooltip 详情面板不变
- 保持 暗色/亮色双主题适配

尺寸建议：
- 24px：与工具按钮完全一致，紧凑
- 28px：稍大更易读百分比，视觉略有区分（推荐）

颜色过渡： 从绿色→黄色→红色的变化，让用户一眼感知上下文用量状态，无需 hover 查看详情。
