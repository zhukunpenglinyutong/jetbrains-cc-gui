📋 主题适配检查清单

✓ 图标可见性
  • 暗色主题：使用 --muted (#a8adb7) 作为默认色
  • 亮色主题：使用 --muted (#666666) 作为默认色
  • Hover 状态：切换到 --text 颜色
✓ 对比度要求
  • 普通文本/图标：至少 4.5:1
  • 大号图标 (18px+)：至少 3:1
  • 强调色在背景上：至少 3:1
✓ 特殊图标处理
  • 发送按钮：白色图标 + 彩色背景
  • 状态图标：使用语义颜色
  • 禁用状态：降低透明度至 0.4
