优化建议
统一图标方案
建议使用 Lucide Icons 或自定义 SVG 图标集，替代 Codicon。SVG 图标可以精确控制粗细、圆角等细节，视觉效果更精致。
保持视觉一致性
所有图标应使用相同的 stroke-width (1.8-2px)、stroke-linecap (round) 和 stroke-linejoin (round)，确保视觉风格统一。
图标尺寸规范
ContextBar 工具按钮建议 18px，选择器内图标 14px，发送/停止按钮 16-18px。保持合理的视觉层次。
交互反馈优化
增加 hover 状态的颜色变化和轻微的 scale 动画 (transform: scale(1.05))，提升操作反馈感。发送按钮可增加点击涟漪效果。
布局紧凑化
选择器按钮可以更紧凑，减少 padding。当前的选择器间距较大，可以优化为更紧凑的布局，节省空间。
性能优化
SVG 图标内联可以减少网络请求，但要注意代码体积。建议使用 SVG sprite 或按需加载，避免所有图标一次性加载。
