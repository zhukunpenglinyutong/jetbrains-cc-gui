4. 设计说明

问题：
权限/模式选择按钮（Agent、Plan、Auto）使用 .mode-active-highlight 类，带有独立的 background: rgba(255,255,255,0.04) 和底部 ::after 蓝色 underline，视觉上看起来像独立的外边框/选中态，与旁边其他 selector-button（纯透明背景）风格不统一。

方案 A — 去掉独立背景和 underline 推荐
- 移除 mode-active-highlight 的 background 属性
- 移除 ::after 伪元素的蓝色 underline
- Mode 按钮与其他 selector 按钮视觉完全一致，仅靠文字颜色区分
- hover 时用相同的 rgba(255,255,255,0.06) 背景反馈
- 改动量最小，不影响其他组件

方案 B — Pill 胶囊分组
- 新增 将所有模式选项（Chat/Plan/Agent/Auto）合并为一个胶囊组
- 选中项用 rgba(0,120,212,0.12) 背景高亮
- 一目了然地看到所有可选模式
- 适合宽度足够的场景，窄屏需要回退方案 A

改动文件：
- selectors.css：修改 .mode-active-highlight 和 ::after
- 或 ModeSelect.tsx：改用新的 pill group 组件
