🎨 底部按钮样式 10 种方案

输入区底部 Selector 按钮（Mode/Model/Reasoning）的视觉风格

① Flat Tab + Underline — 当前样式

透明背景 + 选中项底部蓝色线条。最常见于 IDE 工具栏。改动量为零。
35

② Border Outline 方框描边

选中项用 1px 实线边框 + 微背景。经典 macOS 风格，清晰但不抢眼。

③ Pill Capsule 胶囊

border-radius: 14px 全圆角。现代感强，常见于 Notion / Linear 风格产品。

④ Oval Tag 椭圆标签

padding 横向加大 + border-radius: 100px，更紧凑的"标签"质感。常见于 GitHub 标签。

⑤ Soft Shadow 柔和阴影

选中项加柔和 box-shadow，无边框。iOS/macOS 常用的选中态。

⑥ Ghost Transparent 幽灵按钮

所有按钮完全透明，hover 才显示背景。极简风，适合工具密集型界面。

⑦ Inset Pressed 内陷按压

选中项用 inset box-shadow 模拟按压态。物理按钮质感，常见于 VS Code 面板。

⑧ Accent Dot 强调圆点

选中项顶部加小圆点指示器。Telegram/Slack 风格，占用空间最小。

⑨ Icon Only 圆形图标按钮

去掉文字只保留图标，hover 弹出文字。最紧凑，适合窄面板。

⑩ Segmented Control 分段控制器

Mode 选项合并为一个分段控件（iOS 风格）。选中项有独立滑块背景。
