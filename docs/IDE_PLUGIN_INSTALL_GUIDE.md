# IDE 插件安装操作文档

## 1. 适用范围

本文适用于 JetBrains 系列 IDE 的插件安装操作，包括 IntelliJ IDEA、PyCharm、WebStorm、GoLand、PhpStorm、CLion 等。

## 2. 安装前准备

安装插件前，请确认以下事项：

1. IDE 已正常启动。
2. 当前 IDE 版本满足插件要求。
3. 如需在线安装插件，请确认网络可访问 JetBrains Marketplace。
4. 如需本地安装插件，请提前准备好插件安装包，通常为 `.zip` 或 `.jar` 文件。

## 3. 在线安装插件

1. 打开 IDE。
2. 进入插件管理页面：
   - Windows / Linux：点击 `File` -> `Settings` -> `Plugins`
   - macOS：点击 IDE 名称菜单 -> `Settings` -> `Plugins`
3. 在 `Marketplace` 页签中输入插件名称。
4. 在搜索结果中找到目标插件，点击 `Install`。
5. 等待插件下载并安装完成。
6. 如果 IDE 提示需要重启，点击 `Restart IDE`。
7. IDE 重启后，进入 `Settings` -> `Plugins` -> `Installed`，确认插件已出现在已安装列表中。

## 4. 本地安装插件

适用于安装内部插件、离线插件或从指定渠道下载的插件包。

1. 打开 IDE。
2. 进入插件管理页面：
   - Windows / Linux：点击 `File` -> `Settings` -> `Plugins`
   - macOS：点击 IDE 名称菜单 -> `Settings` -> `Plugins`
3. 点击页面右上角的齿轮图标。
4. 选择 `Install Plugin from Disk...`。
5. 在文件选择窗口中选择插件安装包。
6. 点击 `OK` 或 `Open` 开始安装。
7. 如果 IDE 提示需要重启，点击 `Restart IDE`。
8. IDE 重启后，进入 `Settings` -> `Plugins` -> `Installed`，确认插件已安装并处于启用状态。

注意：本地安装 `.zip` 插件包时，一般不需要手动解压，直接选择插件包文件即可。

## 5. 启用或禁用插件

1. 打开 `Settings` -> `Plugins`。
2. 切换到 `Installed` 页签。
3. 在插件列表中找到目标插件。
4. 勾选插件名称右侧的复选框可启用插件；取消勾选可禁用插件。
5. 如果 IDE 提示需要重启，按提示重启 IDE。

## 6. 卸载插件

1. 打开 `Settings` -> `Plugins`。
2. 切换到 `Installed` 页签。
3. 找到需要卸载的插件。
4. 点击插件右侧的更多操作按钮，选择 `Uninstall`。
5. 确认卸载操作。
6. 如果 IDE 提示需要重启，按提示重启 IDE。

## 7. 安装后验证

插件安装完成后，可以通过以下方式确认插件是否生效：

1. 在 `Installed` 插件列表中确认插件存在且已启用。
2. 查看 IDE 菜单栏、工具窗口或右键菜单中是否出现插件相关入口。
3. 打开一个项目，确认插件功能可以正常使用。
4. 如插件需要登录、配置服务地址或填写 Token，请先完成插件配置。

## 8. 常见问题

### 8.1 搜索不到插件

可能原因：

1. 网络无法访问 JetBrains Marketplace。
2. 插件名称输入错误。
3. 当前 IDE 版本不兼容该插件。
4. 插件未发布到 Marketplace，需要使用本地安装方式。

处理方式：

1. 检查网络和代理配置。
2. 确认插件名称是否正确。
3. 确认 IDE 版本是否满足插件要求。
4. 向插件提供方获取本地安装包。

### 8.2 安装后没有生效

可能原因：

1. 插件未启用。
2. 安装后未重启 IDE。
3. 插件与当前 IDE 版本不兼容。
4. 插件需要额外配置后才能使用。

处理方式：

1. 进入 `Settings` -> `Plugins` -> `Installed`，确认插件已启用。
2. 重启 IDE。
3. 检查插件详情中的兼容版本说明。
4. 根据插件使用说明完成必要配置。

### 8.3 本地安装失败

可能原因：

1. 插件包文件损坏。
2. 插件包格式不正确。
3. 插件版本与当前 IDE 不兼容。

处理方式：

1. 重新下载插件包。
2. 确认插件包为 `.zip` 或 `.jar` 格式。
3. 确认插件包没有被手动解压或改名。
4. 使用与当前 IDE 版本匹配的插件包重新安装。

## 9. 注意事项

1. 安装未知来源插件前，请确认插件来源可信。
2. 升级 IDE 前，建议确认关键插件是否支持新版本 IDE。
3. 多人统一使用插件时，建议明确插件版本，避免因版本不一致导致功能差异。
4. 插件异常时，可先禁用插件并重启 IDE，确认问题是否由插件引起。
