💡 技术优化说明

高帧率适配关键点：

  • 使用 cubic-bezier(0.4, 0, 0.2, 1) 缓动函数替代线性动画
  • 添加 will-change 提示浏览器优化动画性能
  • 优化动画关键帧，确保在60fps下平滑过渡
  • 使用 transform 和 opacity 进行动画，避免触发布局重排
  • 为旋转、缩放等动画添加硬件加速提示
