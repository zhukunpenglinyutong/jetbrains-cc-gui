📐 60帧动效规则

❌ 禁止动画的属性

  • box-shadow — 每帧重绘像素区域
  • filter: blur() — 高斯模糊，GPU 杀手
  • background — 触发整元素重绘
  • border-color — 触发重绘
  • width / height — 触发布局重排
  • margin / padding — 触发布局重排
  • top / left — 触发布局重排

✅ 只用这两个属性做动画

  • transform — GPU 合成器线程处理
  • opacity — GPU 合成器线程处理
  • 替代技巧：
  • 背景色过渡 → 伪元素 + opacity
  • 发光效果 → 伪元素 border + opacity
  • 展开收起 → scaleY 或 clip-path
  • 进度条 → scaleX（非 width）
必加声明： 对所有动画元素加 will-change: transform, opacity;（提前提升合成层）
布局隔离： 对卡片/列表项加 contain: layout style;（避免影响邻居布局计算）
